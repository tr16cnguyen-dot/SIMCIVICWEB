# SIMCIVIC — Where Leaders Are Made

The official landing page for **SimCivic**, a PBL city and nation simulation where students manage economies, forge alliances, navigate diplomacy, and build thriving cities in real time.

## Live Site

Hosted via GitHub Pages at: `https://tr16cnguyen-dot.github.io/SIMCIVICWEB/`

## Dashboard

The interactive teacher/student scoreboard lives in a separate repo: [SimCivic](https://github.com/tr16cnguyen-dot/SimCivic)

## Project Structure

```
index.html   — single-file landing page (HTML + CSS + JS, no build step)
LICENSE      — MIT
SECURITY.md  — vulnerability reporting policy
.gitignore   — standard web ignores
```

## Running Locally

No build tools needed. Just open `index.html` in a browser, or serve with any static file server:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```
